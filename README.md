# Design-and-Verification-of-a-6T-SRAM-Cell-Using-Cadence-Tools
Design and verification of a 6T SRAM cell using Cadence Virtuoso in CMOS technology. Includes schematic, layout, DRC, LVS, RCX, transient simulations for read '0' and read '1' operations, and power analysis. Demonstrates correct functionality, design-rule compliance, and efficient low-power SRAM implementation for VLSI applications.
